The formula we use to perform linear regression is:
$Y = mX + a$
where:
Y → path
m → the slope of our line. It is calculated as such: $r\frac{S_{y}}{S_{x}}$
X → the speed of movement
a → $mean(y) - m * mean(x)$

The scatter plot clearly shows that a linear regression model with the raw data of path will prove to be innacurate. There are outliers present and they have an undue influence on the regression line. In order to fix this, we may take the square root of the path values.

Testing our linear model

At first, we need to figure out whether we have a correlation between our X and Y values. This can be done by checking if the Null Hypothesis (H0) is valid. We may use a t-test to accomplish this. Let us assume that $\alpha = 0.05$.

In [131]:
n = len(hitrost)
alpha = 0.05
cv = stats.t.ppf(1.0 - alpha/2, n - 2)

T = (b1 / std_res) * np.sqrt((n - 1) * np.std(hitrost))
t_stat, p_val = ttest_ind(hitrost, pot_sqrt)

print(f"The t-statistic:{t_stat: .4f}, the critical value(cv):{cv: .4f}, and the test: {T: .4f}")

The t-statistic: 13.3378, the critical value(cv): 2.0003, and the test:  6.8610

Since $|t_{statistic}| \geqslant c_v$ we can ignore the null hypothesis ($H_0$). This means that there is a relationships between our two variables. Knowing this, we set out to find the correlation.
